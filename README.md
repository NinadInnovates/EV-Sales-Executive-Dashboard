# EV-Sales-Executive-Dashboard

Project Overview:

This project presents an interactive EV (Electric Vehicle) Executive Dashboard built using Power BI.
The dashboard analyzes sales performance across different states, years, and vehicle categories to derive actionable business insights.

Objective:

The main objective of this project is to:

   1. Analyze EV sales trends over time
   2. Identify top-performing states
   3. Understand vehicle category distribution
   4. Provide insights to support business decision-making

Dashboard Features:

   1. Total EV Sales
   2. Year-over-Year (YOY) Growth
   3. Top Performing State
   4. Top Vehicle Category

Key Insights :

  1. EV sales have grown significantly after 2021
  2. Uttar Pradesh is the top-performing state
  3. 2-Wheelers contribute ~50% of total EV sales
  4. Sales are concentrated in a few states
  5. Seasonal peaks observed during Nov–Jan
  6. Slight decline in 2024 (possible incomplete data)

Tools & Technologies Used:

  1. Power BI (Data Visualization)
  2. DAX (Data Analysis Expressions)
  3. Data Modeling
  4. Data Cleaning

Business Impact:

This dashboard helps stakeholders to:

  1. Identify high-growth regions
  2. Understand customer preferences
  3. Plan market expansion strategies
  4. Make data-driven business decisions

Future Improvements:

  1. Add forecasting for future sales trends
  2. Include customer segmentation analysis
  3. Enhance interactivity with drill-through features
  4. Integrate real-time data sources
